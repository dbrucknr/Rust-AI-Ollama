# Rust-AI-Ollama

### Dependencies

- Rust Async Runtime: `cargo add tokio -F tokio/full`
- Base AI Layer: `cargo add rig-core -F rig-core/ollama futures_util`
- Backend API: `cargo add axum serde -F serde/derive`
